Configuring Git

  1. Go to VS code and click Terminal. > new terminal.
  2. Check if git is already installed by typing git --version in the command line.
  3. Configure Git in your local computer.
    • Set your username
      git config --global user.name"Pedro"
    • Set your email address
      git config --global user.email"Pedro@123.com"
  4. To initialize the project directory, type git init

Git add and commit

  1. To see the status of your file in your directory, type git status
  2. To track your files, use the command git add
    Syntax git add. for group files, if you want to 2 or more files / group files
    Syntax git add file name for individual files.
  3. By running the git status again you can see the staged files in green ready for commit.
  4. Commit the files that you have staged in your local repository by typing
    Syntax git commit -m "message".
